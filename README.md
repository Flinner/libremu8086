# Libremu8086

Libremu8086 is a libre - free as in freedom - clean room
implementation of emu8086. It (should) support windows, linux and the
web.

## Installation
## Compile
`Cmake` is required to build this project.

```bash
$ cmake -S . -B build && cmake --build build
```
## Test
```
cd build/
make test
```
